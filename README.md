# Fantasy Battle Simulator
Реализован объектно-ориентированный симулятор сражений фэнтези-существ, предоставляющий полный цикл создания существ, их модификации, размещения на столах игроков и проведения боёв. Система построена на современных принципах ООП и паттернах проектирования.

## Реализованный функционал
### Базовая модель существ (Creature)
- Каждое существо имеет показатели атаки и здоровья (нотация x/y)
- Реализованы базовые операции: атака, получение урона, изменение характеристик

### Каталог существ
- Изменения характеристик существ на столе не влияют на оригиналы в каталоге

### Модификаторы (реализованы как Декораторы)
- Магический щит - однократное избегание любого урона
- Мастерство атаки - возможность провести две атаки подряд
- Комбинирование нескольких модификаторов одного типа
- Динамическое добавление/удаление модификаторов

### Заклинания (постоянные улучшения)
- Зелье силы - увеличение атаки на 5 единиц
- Зелье выносливости - увеличение здоровья на 5 единиц
- Амулет защиты - применение Магического щита
- Волшебное зеркало - обмен показателями атаки и здоровья

### Конкретные типы существ
- Боевой аналитик (2/4) - увеличивает атаку на 2 перед каждой атакой
- Злобный боец (1/6) - удваивает атаку при получении не смертельного урона
- Сундук-мимик (1/1) - копирует максимальные показатели противника
- Бессмертный Ужас (4/4) - однократное воскрешение с 1 HP
- Мастер амулетов (5/2) - изначально имеет Магический щит и Мастерство атаки

### Столы игроков
- Поддержка до 7 существ на столе
- Изоляция изменений между столами разных игроков
- Операции получения атакующих и атакуемых существ

### Система боя
- Поочередные атаки существ игроков
- Определение победителя/ничьей

## Реализованные паттерны проектирования
### Порождающие паттерны
- Строитель (Builder) - для пошагового создания сложных объектов существ с модификаторами
- Шаблонный метод (Template Method) - в иерархии строителей для стандартизации процесса создания
- Прототип (Prototype) - для клонирования существ и столов игроков
- Одиночка (Singleton) - для управления глобальными игровыми сервисами

### Структурные паттерны
- Декоратор (Decorator) - для реализации системы модификаторов существ
- Фасад (Facade) - для упрощения интерфейса проведения боя
